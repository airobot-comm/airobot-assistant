# Contributing to AiRobot Assistant

感谢你对 AiRobot Assistant 的关注！我们欢迎各种形式的贡献，无论是修复 Bug、完善文档、提出新功能，还是分享使用案例。

## 🌟 主要贡献者

- **ronnieshi79** - 项目发起人 & 核心开发者

## 🚀 如何开始贡献

### 1. 发现问题
如果你发现了 Bug 或有改进建议，请先在 [Issues](https://github.com/ronnieshi79/AiRobot-Assistant/issues) 中搜索是否已有讨论。如果没有，欢迎提交一个新的 Issue。

### 2. 开发功能
我们鼓励开发者参与 airobot 功能卡片和服务能力的开发。

- **Fork** 本仓库
- **创建分支**: `git checkout -b feature/your-feature-name`
- **提交代码**: 请遵循项目的 Kotlin 代码规范和架构约定
- **推送分支**: `git push origin feature/your-feature-name`
- **提交 Pull Request**: 向主仓库提交 PR，并详细说明更改内容

## 🛠️ 技术规范

- **编程语言**: Kotlin (遵循官方编码规范)
- **架构**: 遵循项目中的 `doc/architect.md` 定义
- **协作规范**: 遵循项目中的 `doc/rules.md` (Vibe Code 规则)

## 🤝 社区共创

我们正在寻找以下领域的合作伙伴：
- **AI 技术极客**: 优化音频链路、开发创新功能卡片

- **行业解决商**: 针对养老、教育、零售等垂直场景的定制方案
