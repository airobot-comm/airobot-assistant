## 🚀 贡献方法

### 1. 发现问题与建议
如果你发现了 Bug 或有改进建议，请先在 [Issues](https://github.com/ronnieshi79/AiRobot-Assistant/issues) 中搜索是否已有讨论。如果没有，欢迎提交一个新的 Issue。

### 2. 代码贡献流程
1. **Fork** 本仓库至你的 GitHub 组织或用户下。
2. **创建功能分支**：`git checkout -b feature/awesome-your-feature`
3. **提交优化代码**：遵循项目的 Kotlin 代码规范和 Vibe 工程学架构执行。
4. **推送到远程分仓**：`git push origin feature/awesome-your-feature`
5. 向主仓库发起 **Pull Request**。

### 3. 技术规范与约定
- **编程语言**：Kotlin (遵循官方编码规范)
- **架构模式**：遵循项目中的 [`doc/architect.md`](./doc/architect.md) 定义
- **协作流程**：遵循项目中的 [`doc/rules.md`](./doc/rules.md) (Vibe Code 规则)
