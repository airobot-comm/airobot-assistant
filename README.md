# AiRobot Android Tablet

airobot tablet: 多IP角色、全语音交互、主动服务型Ai机器人系统

## 🔠 功能特性
- **角色服务**: 支持基于动画，3D技术的IP型角色互动，可多角色切换
- **语音交互**: 全语音交互，支持语音唤醒，人声识别，功能服务对话
- **主动服务**: ai驱动的功能卡片智能服务，更懂你，主动的服务满足
- **功能卡片**: 按需服务的功能卡片，AI记事/事务闹钟，多媒体播客等

## 🔖 使用说明
- **开发方法**
  - 基于Android SDD + AGP9，下载代码后配置Android Studio环境运行即可

- **使用方法**
  - 启动后配置好网络，语音唤醒即可开始自由对话，功能服务卡片会按需自动弹出
